## Install dependancies
```bash
npm install -y
npm install
```

# Partie Back-end
```bash
cd backend
npm install
npm start
cd..
```

# Partie Front-end
```bash
cd frontend
npm install
npm start
cd..
```

## Deploy backend on CloudFlare

```bash
npm create cloudflare@latest -- --template https://github.com/JJublanc/journey_cooking.git/backend#dev
```
